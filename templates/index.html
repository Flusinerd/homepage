About me

Hi, I'm Jan Krüger, a full-stack web developer from Germany. I'm passionate about building web applications and learning new technologies. Currently working with NestJS, Angular, TailwindCSS and .NET Core.

Projects

These are some of my projects. I'm currently working on a project management app with .NET Core and Angular and helping the NestJS community.

SmartChef

SmartChef is a smart shopping list that utilizes barcodes for item identification. It provides you amount of goods you need to purchase in order to keep the desired stock level.

Farmer Companion

Farmer Companion is a hybrid app that helps farmers to monitor their crops and get information about the weather. It also provides a automationsystem inspired by Home Assistant to control devices like the irrigation system.

ScrumBO

ScrumBO is a scrum board that allows you to manage your projects. It has User Stories, Tasks and Sprints and allows multiple teams to coexist. It is currently in development.
